this 바인딩

this 바인딩은 기본적으로 3가지 유형이 있다.
this는 함수가 호출되는 패턴에 따라 this가 어디에 바인딩 되는지 결정된다.

  1. 전역함수를 호출할 때

  2. 함수가 객체의 속성이고 해당 객체를 통해 함수를 호출할 때

  3. 생성자 함수(new)를 호출할 때

이외의 방법으로 call(),apply(), bind() 메서드가 존재한다.
